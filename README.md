# 📊 Segmentação de Clientes FashionLux: Análise de Clusterização K-Means
Este projeto aplica técnicas de Machine Learning e Análise de Dados para segmentar a base de clientes da "FashionLux", uma empresa de varejo de moda nos EUA. O objetivo é identificar padrões de comportamento de compra para direcionar estratégias de marketing personalizadas.

🎯 Objetivo do Projeto
Agrupar os clientes em 4 segmentos distintos com base em comportamento de compra, demografia e preferências, permitindo que a FashionLux:

- Personalize campanhas de marketing.
- Otimize a gestão de estoque por categoria.
- Ajuste as opções de pagamento e frequência de ofertas.


🛠️ Tecnologias e Métodos
- Linguagem: Python (Pandas, Scikit-Learn, Matplotlib/Seaborn)
- Algoritmo: K-Means Clustering
- Técnicas: Pré-processamento de dados, Padronização (Scaling), Análise de Distribuição Geográfica e Demográfica.


🚀 Impactos Estratégicos (Marketing & Negócios)
A partir da análise dos clusters, as principais oportunidades para a FashionLux são:

- Marketing Personalizado: Criação de campanhas específicas para cada perfil, aumentando a conversão e o engajamento.

- Expansão Geográfica: Foco de investimentos e abertura de lojas físicas nos 4 estados de maior densidade de clientes.

- Otimização de Conversão: Expansão dos métodos de pagamento (como Venmo) para reduzir o abandono de carrinho.

- Fidelização e LTV: Implementação de programas de fidelidade segmentados para aumentar o tempo de vida do cliente com a marca.

- Ajuste de Mix de Produtos: Alocação de estoque baseada nas categorias preferidas de cada grupo, evitando desperdícios.
